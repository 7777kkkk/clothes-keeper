## 一、项目概览

### 1.1 项目信息
- **名称**：衣柜APP（ClothesKeeper）
- **平台**：React Native (Expo)，iOS & Android
- **目录**：`/home/alluki/clothes-keeper`
- **仓库**：`https://github.com/7777kkkk/clothes-keeper`

### 1.2 技术栈
- **框架**：Expo + React Native + TypeScript
- **状态管理**：Zustand
- **本地存储**：AsyncStorage
- **导航**：React Navigation 6（底部 Tab + Stack）
- **图标**：Ionicons

### 1.3 核心痛点
1. 衣物管理难
2. 穿搭决策难
3. 录入成本高

## 二、最高优先级：日历页全量重构

### 2.1 顶部导航栏
- **压缩高度**：符合移动端标准（44-48pt）
- **布局**：
  - 左侧：返回按钮（如有上级页面）
  - 中间：月份在上（较大字号）、年份在下（较小字号）紧凑排布
  - 右侧：功能图标（今日、更多选项）
- **月份下拉**：支持点击月份弹出年月选择器

### 2.2 日历网格核心重构
- **布局规范**：
  - 7列均分屏幕宽度
  - 星期栏与日期栏严格上下对齐
  - 日期格子为近正方形（行高=列宽）
  - 格子为圆角卡片，横向纵向间距均匀
- **内容规范**：
  - 日期数字固定左上角，不遮挡缩略图
  - 当月日期：主文字色
  - 跨月日期：浅灰色
  - 今日：「今」字专属高亮标记
  - 选中日期：明显边框/底色高亮
  - 有穿搭记录的日期：格子内居中展示穿搭缩略图
    - 单件：单图展示
    - 多件：并排等比例缩小展示
- **交互规范**：
  - 点击日期格子自动选中
  - 下方穿搭详情卡片同步切换
  - 支持左右滑动切换月份，动画流畅

### 2.3 穿搭详情卡片
- **布局**：
  - 左右边距与日历网格统一
  - 内边距规范均匀
- **左上角**：「选中日期 + 天气图标 + 实时温度」
  - 天气数据对接真实API
- **右上角**：单一个编辑/更多功能入口（删除重复按钮）
- **穿搭单品展示**：
  - 等比例圆角大图横向均分排列
  - 与日历格子内缩略图一一对应
- **右下角**：「+」悬浮按钮
  - 位置在安全区域内
  - 与底部导航栏保持足够距离
  - 点击可新增当日穿搭

### 2.4 底部导航栏
- 统一所有tab图标尺寸
- 图标与文字上下间距均匀
- 选中态：主色高亮
- 未选中态：中性灰色
- 清晰区分当前页面

## 四、核心功能要求

### 4.1 极简衣物录入模块（目标功能）
- AI自动录入：
  - 拍照/上传图片后自动完成抠图
  - 自动识别衣物类型、颜色、材质
  - 自动填充字段
- 批量录入支持
- 电商订单同步导入
- 同款衣物一键去重

### 4.2 智能搭配核心模块（目标功能）
- 自由搭配试验：
  - 衣物自由组合
  - 拖拽排序
  - 图层调整
  - 一键保存
  - 一键标记为当日穿着
- 场合/季节/温度筛选适配衣物
- 基础搭配逻辑校验
- 搭配建议推送

### 4.3 数据导入导出
- **导出**：导出为JSON文件（非文本）
- **导入**：从JSON文件导入
- 文件格式：`clothes-keeper-backup-YYYY-MM-DD.json`

### 4.4 场景化联动功能（目标功能）
- 对接天气API，根据气温/天气推荐穿搭
- 对接手机日历，根据日程场合推送穿搭建议
- 衣物换季收纳提醒
- 洗涤护理提醒

## 七、UI设计参考

### 7.1 色彩系统（基于无忧衣柜）
```
主色(primary):      #4A90D9
主色深(primaryDark): #3A7BC8
强调色(accent):     #5AC8FA
成功色(success):     #4CAF50
警告色(warning):     #FFB74D
错误色(error):        #FF6B6B

背景色(background):   #F5F7FA
卡片背景(card):       #FFFFFF
主文字(textPrimary): #333333
次文字(textSecondary): #666666
占位文字(placeholder): #999999

边框色(border):      #E8E8E8
分割线(divider):      #F0F0F0
```

### 7.2 字体规范
```
系统字体：iOS: SF Pro / Android: Roboto
页面大标题: 20px, Bold
模块标题: 17px, SemiBold
正文: 15px, Regular
辅助文字: 13px, Regular
标签文字: 11px, Medium
```

### 7.3 间距规范
```
基础单位: 4px
页面左右边距: 16px
卡片内边距: 16px
元素间距(小): 8px
元素间距(中): 12px
元素间距(大): 16px
区块间距: 24px
```

### 7.4 圆角规范
```
小圆角(sm):  8px  （标签、小按钮）
中圆角(md):  12px （卡片、输入框）
大圆角(lg):  16px （大卡片、弹窗）
全圆角(full): 9999px（药丸按钮）
```

### 7.5 阴影规范
```
卡片阴影:  0 2px 8px rgba(0,0,0,0.08)
浮层阴影:  0 4px 16px rgba(0,0,0,0.12)
FAB阴影:   0 6px 20px rgba(0,0,0,0.16)
```
